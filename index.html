NPM Translate

NPM translate facilitates the managements of the translation files for developers and provides a simple tool for the community to assist in editing translations of their favorite sites.

Go to the App

What is this?

For developers

An easy way to integrate translations and get them back with NPM @scope.

For translators

An app to contribute at the translations of your favorites websites or applications.

How it work?

For developers

  • Log in the app
  • Create a package for your translations
  • Submit your translations key list in the app
  • Valid the translations of the community
  • Publish the translation on NPM

For translators

  • Log in the app
  • Select a package you want to help to translate
  • Translate the missing filds in your language or purpose a better translation
